4. **Remove quarantine attribute (Required for first-time setup):**

   ```bash
   xattr -d com.apple.quarantine "/Applications/G Smart Translator.app"
   ```

## ğŸ“– Usage Guide

### First Time Setup

1. **Login:**

   - Use the username provided in your approval email
   - First-time login on a new device requires approval

2. **Configure Settings:**
   - Go to **Settings** in the navigation menu
   - Enter your **OpenAI API key** (required for translation)
   - Optionally set a default **translation context** (e.g., "E-commerce Website", "Mobile Banking App")
   - Optionally set a default **translation folder path**

### Using the App

1. **Select Folder:**

   - Click "Home" in the navigation
   - Click "Select Folder" and choose the folder containing your JSON translation files

2. **Choose Reference File:**

   - Select the base language file (e.g., `en.json`, `fr.json`)

3. **Select Target Files:**

   - Choose which language files need translation or comparison

4. **Choose Mode:**

   - **Compare Only** - See missing/extra keys without translating
   - **Remove Extra Keys** - Automatically remove keys that exist in target files but not in the reference file
   - **AI Translate** - Automatically translate missing keys using OpenAI

5. **Start Processing:**
   - Click "Start Processing" and watch the magic happen!
   - The app will show progress and save translated files automatically

### Tips

- The app caches translations to save API costs
- You can customize JSON formatting in Settings
- Context helps the AI provide better translations
- First-time device login requires approval
- Translations are saved automatically to your selected folder
